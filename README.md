# Практична робота: Створення статичних методів
## Обране завдання 23: Розрахунок "Числа долі"
**Виконала Гунченко Влада 35 група**
**Число долі** — це одне з найважливіших чисел у нумерології, яке розраховується на основі повної дати народження.

### Алгоритм розрахунку:
1. Складаються всі цифри дня, місяця та року народження.
2. Якщо сума двозначна, цифри додаються знову.
3. Процес повторюється до отримання числа від 1 до 9.


### Реалізація:
У проекті використано:
* **Алгоритмічна оптимізація**: замість багаторазового додавання цифр у циклі `while`, я використав формулу **цифрового кореня** через оператор залишку від ділення `% 9`.
* **Валідація**: додано базову перевірку вхідних даних на `null`.
* **Javadoc коментарі** для опису логіки коду.
